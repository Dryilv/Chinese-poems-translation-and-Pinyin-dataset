# 中文诗词数据集

## 数据集简介

本数据集收录了 **13,887 条** 中文古典诗歌，涵盖从唐代至清代的经典作品。我们利用大语言模型对每首诗歌进行了拼音标注和英文翻译，并进行了人工抽查审核，确保数据质量。本项目由 **SCMU 新思路实验室** 提供，适用于自然语言处理、文学研究、教育工具开发等多种应用场景。

---

## 数据格式说明

每个数据项以 JSON 对象形式存储，包含以下字段：

| 字段名 | 数据类型 | 说明 |
|--------|----------|------|
| `doc_id` | 整数 | 诗歌的唯一标识符 |
| `title` | 字符串 | 诗歌标题 |
| `author` | 字符串 | 作者姓名 |
| `dynasty` | 字符串 | 所属朝代 |
| `content` | 字符串 | 诗歌原文 |
| `translate` | 字符串 | 英文翻译 |
| `note` | 字符串 | 对诗中特定词语的注释 |
| `analysis` | 字符串 | 对诗歌的整体分析与鉴赏 |
| `pingyin` | 对象 | 拼音标注信息，包含两种格式：<br> - `numeric`：数字格式拼音<br> - `unicode`：Unicode 格式拼音 |
